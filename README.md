# 4-character-captcha-solver

Trains a CNN to predict 4-character captcha text from images.

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install torch torchvision pillow --index-url https://download.pytorch.org/whl/rocm6.2.4
```

> **AMD GPU (gfx1103):** prefix every command below with `HSA_OVERRIDE_GFX_VERSION=11.0.0`

## Workflow

### 1. Collect images

Downloads captchas into `captchas/`, numbered sequentially. Edit `COUNT` at the top to change how many.

```bash
python fetch_captcha.py
```

### 2. Label images

Opens a GUI showing each image. Type the 4 characters you see and press Enter to save and advance.
Labels are written to `labels.csv`. Restarting resumes from the first unlabeled image.

```bash
python label_captchas.py
```

### 3. Train

Trains the model on `labels.csv` and saves the best checkpoint to `captcha_model.pth`.
Accuracy is printed each epoch — aim for 80%+ before using for inference.

```bash
python trainer.py
```

### 4. Predict

```bash
python predict.py captchas/1.png
```

Prints the predicted 4-character string to stdout.

## Files

| File | Purpose |
|---|---|
| `fetch_captcha.py` | Downloads captcha images |
| `label_captchas.py` | GUI labeling tool |
| `trainer.py` | Model definition and training loop |
| `predict.py` | Run inference on a single image |
| `labels.csv` | Image filename → label pairs |
| `vocab.json` | Character index mapping (generated by trainer) |
| `captcha_model.pth` | Saved model weights (generated by trainer) |
